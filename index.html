Dataset

We use Large Movie Review Dataset from Stanford University (http://ai.stanford.edu/~amaas/data/sentiment/) as training dataset.

As for testing dataset, we use Twitter API and Python to retrieve tweets from Twitter by search keywords of a movie. After preprocessing and filtering spam and irrelevant tweets, we use this tweets to test our Bayes Networks using 10-fold cross validation.
